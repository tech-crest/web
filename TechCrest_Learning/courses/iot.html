📚 Course Syllabus

W1
IoT Fundamentals & ESP32 Setup
5 topics · 4 hrs
+
What is IoT? Architecture and real-world use cases
ESP32 vs ESP8266: choosing the right board
Setting up MicroPython and Thonny IDE
GPIO basics: reading sensors and controlling actuators
Project: LED control via smartphone over Wi-Fi
W2
Sensors, Actuators & Data Reading
6 topics · 4 hrs
+
DHT22 temperature & humidity sensor
PIR motion sensor and ultrasonic distance sensor
LDR light sensor and soil moisture sensor
Relay module for controlling AC appliances
OLED display for local data visualization
Project: Home environment monitoring station
W3–4
MQTT Protocol & Node-RED
7 topics · 8 hrs
+
MQTT concepts: broker, topics, QoS levels
Setting up Mosquitto MQTT broker locally
Publishing sensor data from ESP32 to broker
Subscribing and processing messages with Python
Node-RED installation and flow programming
Building a Node-RED dashboard with live charts
Project: Real-time sensor dashboard via MQTT + Node-RED
W5–6
Cloud Integration — AWS IoT Core
7 topics · 8 hrs
+
AWS IoT Core overview and free tier setup
Creating IoT Things, certificates and policies
Connecting ESP32 to AWS IoT over MQTTS
Device Shadow for state management
AWS Lambda functions for IoT data processing
DynamoDB for IoT data storage
Project: Cloud-connected smart plant watering system
W7–8
REST APIs, HTTP & Web Integration
6 topics · 6 hrs
+
HTTP GET / POST requests from ESP32
Building a simple REST API with Flask (Python)
Sending alerts via Telegram Bot API
ThingSpeak for free IoT data logging
Web dashboard using HTML + JavaScript
Project: IoT security camera with motion alerts
W9–10
Capstone Project & Career Prep
5 topics · 8 hrs
+
Capstone: Full IoT system — sensors + cloud + dashboard
System architecture documentation
Portfolio project demo and review
Resume building for IoT / embedded roles
Mock interview: IoT protocols and cloud concepts
