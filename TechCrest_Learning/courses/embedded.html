📚 Course Syllabus

W1
Introduction to Embedded Systems & C Basics
6 topics · 4 hrs
+
What is an embedded system? Real-world examples
Overview of microcontrollers vs microprocessors
Setting up development environment (Keil / STM32CubeIDE)
C programming refresher: data types, pointers, structs
Bitwise operations and register manipulation
First program: Blink an LED on Arduino
W2
GPIO, Digital I/O & Debouncing
5 topics · 4 hrs
+
GPIO registers: input, output, alternate function
Pull-up and pull-down resistors
Reading push buttons and switches
Software debouncing techniques
Project: Button-controlled LED patterns
W3
Timers, PWM & Interrupts
6 topics · 5 hrs
+
Timer modes: input capture, output compare, PWM
Generating precise delays using timers
PWM signals: duty cycle and frequency control
Interrupt service routines (ISR)
NVIC configuration and interrupt priority
Project: Servo motor control via PWM
W4
ADC, DAC & Sensor Interfacing
5 topics · 4 hrs
+
Analog-to-Digital conversion fundamentals
STM32 ADC configuration and DMA transfer
Temperature sensor (LM35 / NTC thermistor) reading
DAC output for waveform generation
Project: Digital thermometer with LCD display
W5–6
Communication Protocols: UART, SPI & I2C
8 topics · 8 hrs
+
UART: baud rate, framing, serial terminal debugging
SPI: master-slave, CPOL/CPHA modes
I2C: addressing, ACK/NACK, clock stretching
Interfacing OLED display over I2C
Reading accelerometer over SPI
GPS module communication over UART
Debugging protocols with logic analyser
Project: Multi-sensor data logger
W7–8
Real-Time Operating Systems (FreeRTOS)
7 topics · 8 hrs
+
RTOS concepts: tasks, scheduler, context switching
FreeRTOS task creation and priorities
Queues and inter-task communication
Semaphores and mutexes
Software timers and event groups
Stack overflow detection and debugging
Project: RTOS-based home automation controller
W9–10
Advanced Peripherals & Power Management
6 topics · 6 hrs
+
Watchdog timer and system reset handling
Flash memory read/write operations
DMA transfers for high-speed data
Low-power modes: sleep, stop, standby
RTC (Real-Time Clock) configuration
Battery-powered embedded device design
W11–12
Capstone Project & Career Prep
5 topics · 8 hrs
+
Capstone project: end-to-end embedded system build
Code review, documentation, and version control (Git)
Portfolio project presentation
Resume building for embedded roles
Mock interview: embedded C + hardware concepts
